## 1. 서비스별 실사 결과

### 채비넷 — 장애자녀평생설계연구소 채비 (chaebee.org / chaebeenet.co.kr)
- **포지션**: 장애자녀 '평생설계' — 부모 부재 이후까지 대비하는 장기 설계. 삼성전자·사랑의열매 사회공헌 기반.
- **평생설계서 8대 영역**: 교육 / 건강 / 여가 / 대인관계 / 직업 / 주거 / 재정 / 법적계획
  — 생애주기별(영유아기~노년기) 목표, 영역별 간편진단 → 종합진단 → 작성 프로세스, AI 도우미, 피쉬본 다이어그램 시각화.
- **돌봄지침서 8개 섹션**: ① 표지(이름·연락처·비상연락처·사진) ② 기본정보(생년월일·장애명·키·체중·**혈액형·의류 사이즈**) ③ 특성(구체적 예시 강조) ④ 일상생활(시간대별 루틴) ⑤ 건강(약물·알레르기·병원) ⑥ 중요 절차(필수 루틴·문제행동 유발 요인) ⑦ **안전(독립 가능 여부·외출 주의·위험 요소)** ⑧ 의사소통(비언어 신호).
- **공유**: SNS(카카오톡) 중심. 모바일앱(채비넷) 병행.
- **강점**: 문서 체계의 깊이, 부모교육·전문가 양성 연계, '내가 없으면?' 프레임.
- **약점**: 작성 부담 큼(진단→학습→작성 단계), 일상(데일리) 기록 없음, 응급 대응 약함, 열람 보안 약함(SNS 공유).

### 자람이 Jarame (jarame.or.kr)
- **포지션**: "발달장애인 AI 디지털케어로그" — 병원·센터·학교·가정이 하나의 기록 공유, 치료·학습 플랫폼.
- **메뉴**: AI 케어로그 / 센터·병원·시설 찾기 / 홈테라피 / 정보자료 / 자가진단 / 커뮤니티(게시판) / 자람이AI(LLM).
- **모델**: 유료 베타(토스페이먼츠). 업데이트 예정: 음성·영상 분석, 바우처 데이터, 행동인식 치료 등.
- **강점**: 기관 통합 비전, AI 브랜딩, 커뮤니티 콘텐츠.
- **약점**: 게시판형 웹 UI(모바일 앱 없음·준비 중), 보호자 혼자서는 가치 약함, 유료 장벽, 핵심 기록 UX 불명확.

### 링크아이 Link-i (Google Play kr.co.linki / App Store id6450511684)
- **포지션**: 발달지연·자폐 아동의 부모-치료사-교사 '팀' 협업 앱 (개발 소규모, 평점 5.0/리뷰 7건).
- **핵심 기능**: ① 아이 코드 공유로 팀 초대(오프라인 선생님을 온라인으로) ② **컨디션 체크(기분·수면·복약 — "효과적 중재의 기초")** ③ **발달검사 기록(여러 기관 결과를 날짜별 저장, 팀 열람 — "여러 번 다시 공유할 필요 없음")** ④ 팀 게시판(중재 목표 논의) ⑤ 링크아이몰(v3.2).
- **강점**: 협업 구조 단순명료, 데일리 컨디션 개념, 검사 기록 보관 발상.
- **약점**: 팀이 모여야 가치 발생(보호자 단독 첫날 가치 약함), 한 장 요약·응급·외부 공유 없음, 규모 영세.

## 2. Stellar Connect 차별화 포인트 (3사 대비)

| 차별점 | 설명 | 3사 보유 여부 |
|---|---|---|
| **한 장 요약 + 응급 우선** | 처음 만나는 사람이 30초에 이해, 비상연락 tel: 링크 최상단 | ✕ |
| **QR 키링 안심 카드** | 가방·키링 부착, 스캔→인증번호→응급 정보 (인쇄 지원) | ✕ |
| **정보 공개 3레벨** | 응급 카드/요약/전체 — 민감정보 통제 + 4자리 인증 | ✕ (채비는 SNS 공유) |
| **입력 부담 최소화** | 카테고리별 추천 칩 원탭, 음성 입력, 한 줄 소개 | △ (채비는 진단·학습 단계가 김) |
| **미아 대비 안심 갤러리** | 최신 사진 보관 + 인상착의·사이즈 | ✕ |
| **보호자 단독 Day-1 가치** | 팀·기관 없이 혼자 시작해도 즉시 유용 | ✕ (링크아이·자람이는 협업 전제) |
| **연구 데이터 정제 설계** | 구조화 입력(상황·강도 카테고라이징) → 메타분석 대비 | △ (자람이 AI는 방향 유사) |

## 3. 벤치마킹 → 이번에 반영한 것 (커밋 기준)

| # | 벤치마킹 출처 | 반영 내용 |
|---|---|---|
| 1 | 채비 돌봄지침서 §2 기본정보 | 아이 신체정보에 **혈액형·의류/신발 사이즈** 추가 → 프로필·응급 블록 표시 |
| 2 | 채비 돌봄지침서 §7 안전 | 설명서 8번째 카테고리 **'안전 주의사항'** 신설(추천 칩 6종 포함) — 응급 카드 공개 레벨에도 포함 |
| 3 | 링크아이 컨디션 체크 | 대시보드 **'오늘의 체크인'** — 기분·수면·식사 원탭 칩 + 복약 체크 통합, 날짜별 저장 |
| 4 | 링크아이 발달검사 기록 | 기록 유형에 **'검사·평가'** 추가 — 여러 기관 검사 결과를 날짜별 보관, 공유로 재제출 대체 |

## 4. 2차 이후 로드맵 후보 (이번 미반영)

- **팀 초대/협업** (링크아이): 공유 코드 기반 열람 → 쓰기 참여(치료사 기록)로 확장. 6/10 회의에서도 '직접 참여는 2차'로 합의.
- **평생설계 모듈** (채비): 8대 영역·생애주기 프레임 차용하되 '진단→작성' 대신 칩 기반 경량 입력으로 차별화. 회의의 '100세 플랜' 요구와 일치.
- **센터·병원 찾기 / 친화 장소 지도** (자람이): 회의 결론대로 후순위 — 제휴 기관 한정 지도부터.
- **AI 케어로그/행동인식** (자람이): 우리 영상 관찰(ABC 태깅) 모듈이 이미 기반 — AI 분석은 아스트로젠 R&D 과제(디지털 치료기기 방향)와 연계.
- **부모 부재 대비 '내가 없을 때' 지침** (채비): 양육자 정보에 대체 돌봄자 지정 + 인수인계 노트.
