# 내 아이 설명서 (Stellar Connect · S:CON) — 포커스 MVP

발달장애 아이를 **누구나 이해할 수 있도록** 돕는 서비스. 매일 기록을 쌓는 앱이 아니라,
부모를 대신해 우리 아이를 설명해 주는 **결과물(대상별 설명서)**을 만들어 주는 것을 목표로 합니다.

> 자람이가 기록을 남기는 서비스라면, 「내 아이 설명서」는 **아이를 이해하게 만드는** 서비스입니다.

## 이 트랙은?

양육자 자문회의·경쟁 서비스 분석 피드백을 반영해, 기능 확장 샌드박스(`stellar-connect-lab`)와
별도로 **포지셔닝에 충실한 포커스 MVP**로 분리한 트랙입니다.

- 라이브: https://hyegeungim3-git.github.io/stellar-connect-mvp/
- 확장 샌드박스(슈퍼셋): https://hyegeungim3-git.github.io/stellar-connect-lab/

## 4대 차별화 기둥

1. **아이를 이해하는 플랫폼** — 좋아하는 것/싫어하는 것, 의사소통 방법, 감각 특성,
   도전적 행동 발생 상황과 효과적 지원 방법을 체계적으로 정리
2. **결과물 = 대상별 「내 아이 설명서」** — 같은 데이터를 **학교용 / 병원용 / 활동지원사용 /
   돌봄기관용**으로 자동 정리해 한 장으로 공유 (반복 설명 부담 해소)
3. **보호자 중심 UX** — 최소 입력 · 빠른 기록(칩 원탭) · 쉬운 공유(QR·링크·안심번호)
4. **생애주기 미래 준비** — 아동기 → 청소년기 → 성인기 → 노년기 자립 플랜

## 화면 (4탭)

홈 · 설명서 · 공유(대상별) · 미래 준비 (+ 양육자/더보기)

## 기술

의존성 없는 순수 HTML/CSS/JS, localStorage 데이터 계층, QR 자체 구현(`assets/js/qr.js`).
`index.html`을 정적 서버로 서빙. 데모 계정 `parent@example.com / 1234`,
관리자 `admin@ichild.kr / admin123`.

기획: 아스트로젠 · 개발: 오큐브 — 1차 MVP 프로토타입.
